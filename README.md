# MoneyRPG
MoneyRPG is an Android app for personal expense tracking, enhanced with simple gamification elements.

## Features
* Record personal expenses
* Local data storage with SQLite
* Lightweight RPG-style progression

## Tech
* Android
* Java
* SQLite
